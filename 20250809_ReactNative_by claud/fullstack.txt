React Native를 프론트엔드로, Node.js (Express)를 백엔드로, MySQL을 데이터베이스로 사용하는 풀스택 로그인 애플리케이션을 개발하려고 합니다.
요구사항은 다음과 같습니다:
0. javascript은 typescript로 작성
1. [기능 요구사항]
  - 사용자는 이메일과 비밀번호로 회원가입 및 로그인할 수 있어야 합니다.
  - 비밀번호는 bcrypt로 해싱해 저장해야 하며, 로그인 시 JWT를 발급받아야 합니다.
  - 로그인된 사용자는 토큰을 통해 인증된 요청을 할 수 있어야 합니다 (Authorization: Bearer).
  - 토큰 만료 및 갱신 로직도 포함해주세요.
2. [보안 고려 사항]
  - 입력값 유효성 검사를 수행해 주세요 (프론트+백엔드).
  - SQL Injection, XSS, 인증 토큰 유효성 등 보안 요소를 고려해 주세요.
  - 비밀번호는 절대 평문으로 저장하지 않습니다.
3. [백엔드 구현 요청]
  - Node.js (Express)로 API를 구현해주세요.
  - API 목록은 회원가입, 로그인, 유저 정보 조회(API: /api/user/me)로 구성해주세요.
  - JWT 인증 미들웨어도 포함해주세요.
  - DB는 MySQL을 사용하며, 사용자 테이블 구조도 생성 SQL로 보여주세요.
  - 전체 백엔드 프로젝트 구조도 현업 기준으로 설계해주세요 (controllers, routes, models, middlewares 등).
4. [프론트엔드 요청]
  - React Native로 로그인, 회원가입, 마이페이지 화면을 구성해주세요.
  - 사용자 입력값 유효성 검사 (이메일 형식, 비밀번호 길이 등)를 포함해주세요.
  - 로그인 후 JWT 토큰을 AsyncStorage에 저장하고, 이를 헤더에 넣어 API 요청에 사용해주세요.
  - 로그인 실패 시 에러 메시지 처리도 포함해주세요.
5. [기타 요구]
  - CORS 설정 포함
  - .env 파일을 통한 환경 변수 관리
  - 모든 코드에 주석을 달아 주세요 (한국어)
  - 코드 외에 동작 방식도 간단하게 설명해 주세요
요청한 내용을 기반으로 프론트/백/DB 전체를 구성해 주세요.  
모든 코드는 실제 프로젝트에서 복사해서 사용할 수 있도록 현업 기준으로 작성해 주세요.
